Montées de version majeures
###########################

Le schéma ci-dessous décrit le principe général pour effectuer plusieurs montée de version majeures en une seule opération de maintenance. Tous les chemins ne sont pas forcément supportés, veuillez vous reporter au support :term:`VITAM` afin de vous assurer que le chemin que vous prévoyez soit officiellement supporté.
