Vérifications préalables
========================

Tous les serveurs cibles doivent avoir accès aux dépôts de binaires contenant les paquets de la solution  logicielle :term:`VITAM` et des composants externes requis pour l'installation. Les autres éléments d'installation (playbook ansible, ...) doivent être disponibles sur la machine ansible orchestrant le déploiement de la solution.
